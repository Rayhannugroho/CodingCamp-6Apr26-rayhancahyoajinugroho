# 🌟 Life Dashboard - To-Do List Web App

A simple and clean **Life Dashboard Web Application** to help you organize your daily activities.  
Built using **HTML, CSS, and Vanilla JavaScript**, this app provides essential productivity tools in one place.

---

## 🚀 Features

### ✅ Core Features (MVP)
- 🕒 **Dynamic Greeting**
  - Displays current date and time
  - Personalized greeting based on time of day

- ⏱️ **Focus Timer (Pomodoro)**
  - 25-minute countdown timer
  - Start, Stop, and Reset controls

- 📝 **To-Do List**
  - Add, edit, and delete tasks
  - Mark tasks as completed
  - Persistent data using Local Storage

- 🔗 **Quick Links**
  - Save and access favorite websites
  - Stored in Local Storage

---

## ✨ Optional Features Implemented
- 🌙 **Dark / Light Mode**
- 👤 **Custom Name in Greeting**
- 🔤 **Sort Task List (A-Z)**

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS** – Styling  
- **JavaScript (Vanilla)** – Logic & interactivity  
- **Local Storage API** – Data persistence
